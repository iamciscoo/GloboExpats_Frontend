# Modern expat platform

## Overview

This is the frontend for Modern Expat Platform, built with Next.js 15, TypeScript, Tailwind CSS, and shadcn/ui components.

## Deployment

You can deploy the app to any platform that supports Next.js (e.g.
[Vercel](https://vercel.com/) or your own container infrastructure).

## How It Works

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Run the development server with `pnpm dev`
4. Build for production with `pnpm build`
